Qingdao Children’s Oral Health Management Platform

2022.06 – Present

Project Overview

A city-wide cloud platform that unifies oral-health services for children in Qingdao, closing the workflow loop among parents, schools, clinics, government bureaus (Education & Health), and sealant-program teams. Core modules cover digital health records, appointment pooling, full clinical pathways (examination → surgery → report), and quality-control oversight. The system is cloud-deployed and responsive on PC, mobile, and tablet browsers.

Key Responsibilities

  • Built & maintained automation frameworks:
    • Interface: Python + Requests + Pytest + PyMySQL + Openpyxl + Loguru + Allure, 200+ cases, 70 % coverage, 50 % faster regression.
    • UI: Python + Selenium + Pytest + Allure + Loguru, 100+ scripts, 60 % main-flow coverage.
    • Applied layered code, data-driven, Page-Object practices; integrated with Jenkins + Git for CI and e-mail reporting; analyzed reports and iterated.

  • Security: ran AppScan end-to-end, identified XSS & SQL-injection risks.
  • Post-release monitoring and user-feedback triage.
