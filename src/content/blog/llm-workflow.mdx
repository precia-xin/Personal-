1. 整型 `int`  `12` → `int()`
2. 浮点 `float` `3.14` → `float()`
3. 布尔 `bool` `True / False` → `bool()`（首字母大写）
4. 字符串 `str` `"hello"` → `str()`
5. 列表 `list` `[1, 2, 3]` → `list()`
6. 字典 `dict` `{"name":"tudou"}` → `dict()`
7. 元组 `tuple` `(1, 2, 3)` → `tuple()`
8. 集合 `set` `{1, 2, 3}` → `set()`
